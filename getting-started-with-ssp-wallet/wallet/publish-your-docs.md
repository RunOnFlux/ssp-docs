# Create Multiple Wallet

{% hint style="info" %}
You may create multiple wallets for the same chain. This will help you create multiple addresses to serve specific purposes. Each wallet is independent with its own balance and history, and you can give each a name and color to recognize it at a glance.
{% endhint %}
