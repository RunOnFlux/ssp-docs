# Using Your SSP Key For Syncing

{% hint style="info" %}
SSP Wallet requires synching from SSP Key which serves as your security for every activities made from your SSP Wallet. This enables you to have a secure transactions and activities on SSP Wallet and eliminates bad actors intentionally making activities using your SSP Wallet.
{% endhint %}

{% hint style="info" %}
With both apps on v2, syncing usually needs no scanning at all: SSP Wallet sends the request through the relay and it pops up on your SSP Key — including multi-chain **batch sync**, where one approval activates several chains at once. The QR code and manual input remain available as a fallback and for offline use.
{% endhint %}

{% hint style="info" %}
From your SSP Wallet, when the pairing/sync screen shows a QR code, you can scan it using your SSP Key — or copy the wallet's sync data from the manual tab into SSP Key's Manual Input
{% endhint %}

You can also use the fingerprint scanner or face recognition of your device in syncing

{% hint style="warning" %}
**Verify the pairing.** After syncing, both devices show the same **6-word verification code** — scan the wallet's verification QR with your SSP Key, or compare the words. If they differ, stop and re-pair: [Pairing Verification Code](pairing-verification-code.md).
{% endhint %}

Congratulations for synching your SSP Wallet and SSP Key!
